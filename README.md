# Passport-Web


### Envars
```shell
REACT_APP_PASSPORT_API_ENDPOINT_HOSTNAME=
SENTRY_DSN_FRONTEND=
XGRID_LICENSE_KEY=
```

### To init
```shell
make init-linux
or
make init-windows
```

### To run
```shell
1. Have passport-server running on 8086
2. make lb
3. make watch

Connect to http://localhost:5003
```
